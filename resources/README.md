## Resources

Complementary documentation and example assets for Omega Testing.

- `class diagram.md` — high-level domain model.
- `erd.md` — entity–relationship diagram of the backend.
- `db-dictionary.csv` — database dictionary for central-backend.sqlite.
- `endpoints.md` — REST API endpoints overview.
- `ExampleRequest.md` — sample HTTP requests.
- `E-commerce-API-Model.postman_collection.json` — Postman collection for the sample e‑commerce API.
- `Central-Backend.postman_collection.json` — Postman collection for the central backend.

### Testing & mocks

- `Resources/testing-mocks/bugs-dashboard-seed.json` — seed data (≈30 bugs) to populate dashboard KPIs and the bugs heatmap.
- `Resources/testing-mocks/seed-dashboard-bugs.ps1` — PowerShell script to POST all bugs to a given project.
